# F1 Race Forecast

A website that visualizes **race-day weather conditions for Formula 1 Grands Prix**, highlighting how weather can impact racing performance and safety.  
Built and designed by **Zainab Ilyas** as part of Sheridan College’s IxD course: *System Design: Databases and Dynamic Data*.

---

## Features

- View **race-day weather forecasts** for Formula 1 Grands Prix  
- See key weather factors including **air temperature, rainfall, humidity, and wind speed**
- Weather conditions categorized into **four impact stages**:
  - **Green** — ideal racing conditions  
  - **Yellow** — early warning  
  - **Orange** — serious race impact  
  - **Red** — dangerous conditions
- Clear summaries explaining **how each condition affects the race**
